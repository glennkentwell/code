# Pascal

High school Turbo Pascal experiments (~1990s).

## Overview

263+ source files (.PAS) with compiled executables (.EXE). Legacy code preserved for historical/nostalgic purposes.

## Structure

```
pascal/
├── *.PAS          # Source files
├── *.EXE          # DOS executables
├── SOURCE/        # Additional source
├── UNITS/         # Reusable units
└── done/          # Completed projects
```

## Notable Programs

| File | Description |
|------|-------------|
| `3DBALL.PAS` / `3DCUBE.PAS` | 3D graphics experiments |
| `256COLOR.PAS` | VGA color palette demos |
| `ANIMALS.PAS` | Animal guessing game |
| `FLITESIM.EXE` | Flight simulator |
| `LIFE.EXE` | Conway's Game of Life |
| `MOLECULE.PAS` | Molecular visualization |
| `MOUSE.PAS` | Mouse input handling |
| `AI.PAS` | AI experiment |

## Requirements

- DOSBox or Turbo Pascal 7.0 environment
- VGA graphics mode support for graphical programs

## Notes

- Executables are 16-bit DOS programs
- Source uses Turbo Pascal-specific units (Crt, Graph, etc.)
- **Do not modify** unless specifically requested—historical preservation
